# COVID-19 Data Analysis - Pakistan (2020)

This project analyzes the COVID-19 pandemic situation in Pakistan during the year 2020 using Python and Pandas.

## Features
- Data Cleaning and Preprocessing (handling missing values, formatting dates)
- Regional Breakdown of COVID-19 Cases (Punjab, Sindh, KP, Balochistan, ICT, etc.)
- Data Visualization of daily trends and cumulative metrics

## Files in this Repository
- `COVID19_2020_PAKISTAN (2).ipynb`: Jupyter Notebook containing the analysis code and visualizations.
- `covid-19-cases-in-pakistan-2020.csv`: The raw dataset used for the analysis.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
